# Persona PDFs

Place your persona PDF files in this directory.

## Access URLs

Files placed here will be accessible at:
```
/portfolio/personas/[filename].pdf
```

## Example

If you place a file named `sovereign-personas-tech-buyer.pdf` in this folder, it will be accessible at:
```
https://saren.ai/portfolio/personas/sovereign-personas-tech-buyer.pdf
```

## Recommended File Naming

- Use lowercase
- Use hyphens instead of spaces
- Be descriptive but concise
- Examples:
  - `persona-enterprise-cto.pdf`
  - `persona-mid-market-director.pdf`
  - `persona-startup-founder.pdf`
